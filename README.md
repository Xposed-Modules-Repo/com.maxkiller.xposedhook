# MaxKiller

Xposed модуль для удаления слежки,телеметрии,ограничения доступа к геолокации,контактам,галерее,камере,микрофону,буферу обмена,другим сенсорам(акселлерометр и тп).

**А также добавляет возможность простого e2e шифрования** чатов через установку одинакового пароля у обоих собеседников,перехватывает отправляемый текст и шифрует его до отправки на сервера

## Требования

* Android 7+
